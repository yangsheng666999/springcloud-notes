## 一、服务注册与发现(Eureka)

Eureka是一个**注册中心**，他的主要作用：

- 微服务启动时向它**注册自己的地址和端口**
- 其他服务可以从它那里**发现目标服务**
- 提供服务健康检查和心跳机制
